# TV Show Tracker React Vite

TV Show Tracker is a React + Vite frontend application to keep a running list of shows to watch or shows you have already watched. It is used in conjuction with the `tv-show-tracker-nodejs` application.

### Tech Stack

- React with Typescript as the JS library
- Vite for running and building the application
- MUI for styling and components (fields, icons, datepickers, tables, etc.)
- Axios and React Query for server requests
- Various other React libraries (`react-router`, `react-hook-form`, `react-toastify`, etc.)

## Build Instructions

#### Set up the .env file

- The `.env.development` file specifies the backend API base URL (such as `localhost:5000`).

#### Run and build the app

- Use `npm run dev` to run the app for development.
- Use `npm run build` to build the app for deploying.
